ANE初始化
--------
		AneSDK.initSDK(isSimulator:Boolean, anePlatform:String)
		isSimulator:是否为模拟器，模拟器传true,真机传false
		anePlatform:目标平台,AneSDK.ANE_PLATFORM_ANDROID,AneSDK.ANE_PLATFORM_IOS
